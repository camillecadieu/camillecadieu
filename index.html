Le Web et ses infrastructures

le web, c.est aussi le Cloud, les Bases de données et les cookies. Les données du web comme les sites, les fichiers et les images sont tous stockés dans des data center aussi appelé base de données avec à l'intérieur des serveurs. C'est pour cela que nous pouvons voir des problèmes avec certains sites quand le data center de Paris nommé Paris Digital Park qui possède plus de 7 hectares de serveurs. Ce que les gens apellent le cloud, c'est en réalité des serveurs qui stockent vos informations. Par exemple Onedrive est un cloud qui stocke vos informations et vos fichiers dans un serveur qui vous est dédié et qui permet de ne pas perdre vos informations si jamais vous les aviez stocké sur votre appareil. Sur la photo n°1, nous pouvons voir le Paris Digital Park vu de l'extérieur et sur la n°2, nous pouvons voir le data center de San José de l'intérieur. Vous vous demandez peut être quesqu'un serveur ? Un serveur est le disque qui est connecté a internet. Donc un serveur est dans le data center et ce sont ces fameux serveurs qui vous traduisent du SQL vers le html que votre ordinateur lit comme dans le schéma ci dessous. On peut voir sur les 2 shémas le trajet d'une requête sur internet. Qu'est ce que le SQL ? Le SQL est le language infomatique que votre ordinateur envoie en requête sous forme de paquet au serveur. Une fois que le serveur a trouvé ce que vous recherchez, alors à ce moment là le serveur va vous renvoyer une requête en SQL qui sera automatiquement traduite par votre ordinateur. les cookies, ces fameuc cookies quz nous acceptons sans reflechir mais savez-vous réellement ce que c'est ? les cookies cest des fichiers quu s.accumulent sur notre appareil, et qui vont servir a notre appareil qui ciblent nos centres d'intérêt. Le RGPD, qu'est-ce que c'est ? Le RGPD c'est Le règlement général de protection des données (RGPD) c'est donc un texte réglementaire européen qui encadre le traitement des données de manière égalitaire sur tout le territoire de l'Union européenne (UE).
